# downloadModel()

Downloads a Whisper.cpp model to the browser.  

```tsx title="app.tsx"
import { useEffect, useState } from "react";
import {downloadModel} from '@remotion/whisper-wasm';

export default function MyApp(){
    const [progress, setProgress] = useState(0)
    useEffect(()=>{
    downloadModel('tiny.en', setProgress);
  },[]);

  return (
    <h1>my great app</h1>
  )
}
```

## Arguments


### `model`

The model to download. Possible values: `tiny`, `tiny.en`, `base`, `base.en`, `small`, `small.en`, `medium`, `medium.en`, `large-v1`, `large-v2`, `large-v3`, `large-v3-turbo`.

### `onProgress?`

Act upon download progress. This is the function signature:

```tsx twoslash

const onProgress = (progress: number) => {
  // do something
};
```


## Return Value

resolves with the message  `Model loaded successfully` if the model has been successfully stored in the `indexedDB` storage of the browser.
