A [reader](/docs/media-parser/readers) for [`@remotion/media-parser`](/docs/media-parser) that reads from a URL using [`fetch()`](https://developer.mozilla.org/en-US/docs/Web/API/Fetch_API).  
It works in the browser, Node.js and Bun.  
This is the default reader for [`@remotion/media-parser`](/docs/media-parser), so you don't need to explicitly specify it.

## Example

```tsx twoslash title="Reading a local file"
import {parseMedia} from '@remotion/media-parser';
import {fetchReader} from '@remotion/media-parser/fetch';

const result = await parseMedia({
  src: 'https://example.com/my-video.mp4',
  fields: {
    durationInSeconds: true,
    dimensions: true,
  },
  reader: fetchReader,
});
```
