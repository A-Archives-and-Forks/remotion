# Recording a scene

To record a scene, run:

```tsx
bun run dev
```

A Vite app will start. Open the URL in the browser.

Give access to the webcam and microphone.

In the top left panel, select the right webcam and microphone.  
In the top right panel, select the screen you want to record.

## Selecting a take

After you have recorded a scene and you are happy with your take, run

```bash
node copy-from-downloads.mjs
```

from the terminal to copy the file from your Downloads folder into your project.
There, the recordings are located under `public/your-composition-id`.
