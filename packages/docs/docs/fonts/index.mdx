The `@remotion/fonts` package provides an API for [loading local fonts](/docs/fonts) into a Remotion component.

## Installation

<Installation pkg="@remotion/fonts" />

## Example Usage

Put the font into your `public/` folder.
To load a font, import the package `@remotion/fonts` and call [`loadFont()`](/docs/fonts/load-font).

```tsx twoslash
import { loadFont } from "@remotion/fonts/";
loadFont({
  family: "some name",
  url: "someName",
  weight: "500",
  // ... other options
}).then(() => console.log("Font loaded!"));
```
